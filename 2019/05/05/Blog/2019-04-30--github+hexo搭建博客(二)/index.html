2019.4.30 GitHub+hexo博客搭建（二）

Created at 2019-05-05 Updated at 2019-05-05 Category Blog Tag Blog

hexo主题修改

我自己的主题是Random
可以根据自己的喜好在官网下载：https://hexo.io/themes/

主题修改

首先进入你想用的主题github仓库，拷贝仓库地址.

在博客根目录打开cmd（以管理员身份运行，否则可能运行不成功）
执行 cd themes ,开始下载

下载完成后，在博客根目录打开文件 _config.yml （站点配置文件） 进行编辑，将theme后的主题改为下载的主题文件名

接下来执行
hexo clean
hexo g
hexo d
打开自己的博客页面，主题更换成功

功能更新

打开themes文件中的主题文件夹，打开_config.yml （主题配置文件）

1. 添加头像

avatar：后面即头像链接，
选择一个自己喜欢的头像生成链接即可

2. menu设置

根据自己的需求更改

3. 社交账号

可自行添加更改，将账号链接放在双引号中间

点击tags或categories或about 的404报错问题

博客的上传都保存于source文件中的_post目录下，文件后缀名为 .md
用hexo命令建一个页面即可

输入命令后会在博客目录source文件下生成新的文件夹

文件夹下分别会有一个 index.md文件
然后设置页面类型

tags

categories

about
